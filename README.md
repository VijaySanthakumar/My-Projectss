# 🍕 Pizza Sales Dashboard

This project features an interactive Pizza Sales Dashboard built using **Power BI** and **Excel**. It showcases key performance metrics and visual analytics to provide insights into pizza sales performance, trends, and customer preferences.

## 🔍 Project Overview

The dashboard analyzes sales data to answer essential business questions, such as:

* Which pizza categories and sizes are the most popular?
* What are the best-performing days and months for sales?
* Who are the top-selling pizza types?
* What is the distribution of revenue, quantity, and orders?

## 📊 Tools Used

* **Power BI** – for dynamic data visualizations and interactive reports
* **Microsoft Excel** – for initial data processing and a separate dashboard
* **CSV File** – as the original data source

## 📁 Files Included

* `pizza_sales.csv`: Raw dataset containing order-level sales data
* `pizza_sales_excel_file.xlsx`: Excel dashboard with charts, KPIs, slicers, and formatted insights
* `pizzasalesreport.pbix`: Power BI report file with interactive visuals and insights

## 📈 Key Features

* One-page dashboards in both Excel and Power BI
* KPIs such as Total Revenue, Total Orders, and Average Order Value
* Slicers for filtering by date, pizza category, and pizza size
* Visuals: stacked column charts, pie charts, line graphs, and heat maps

## 💡 Insights

The dashboard enables stakeholders to quickly identify:

* Top-performing pizzas and categories
* Sales trends over time
* Customer ordering patterns
* Revenue drivers

## 🚀 Getting Started

1. Open `pizzasalesreport.pbix` in Power BI Desktop to explore the report.
2. Open `pizza_sales_excel_file.xlsx` in Excel to view the static version of the dashboard.
3. Use `pizza_sales.csv` as a raw data reference or to build your own custom dashboards.
